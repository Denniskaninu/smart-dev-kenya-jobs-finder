## 📋 Overview

**KenyaDevJobs** is an intelligent job aggregation platform that connects Kenyan developers with legitimate tech job opportunities. Our platform leverages web scraping and AI-powered verification to curate a curated list of authentic job postings, helping developers find their next opportunity with confidence.

## ✨ Key Features

- 🔍 **Smart Job Scraping**: Aggregates tech job postings from multiple sources across the web
- 🛡️ **AI-Powered Fraud Detection**: Uses GenAI technology to identify and filter out fake or suspicious job postings
- 📂 **Smart Filtering**: Filter opportunities by category (Frontend, Backend, Full Stack, DevOps, Data Science, etc.)
- 🔤 **Keyword Search**: Advanced search functionality to find specific roles and technologies
- 📅 **Recent First**: Sort job postings by date to find the latest opportunities
- 🎨 **Clean Interface**: User-friendly design for seamless job discovery

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Denniskaninu/smart-dev-kenya-jobs-finder.git
cd smart-dev-kenya-jobs-finder
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open your browser and navigate to [http://localhost:9002](http://localhost:9002)

## 🤝 How It Works

1. **Scraping**: The platform continuously scrapes job postings from verified tech job boards
2. **Verification**: AI models analyze postings for legitimacy, company credibility, and job authenticity
3. **Aggregation**: All verified jobs are compiled into a unified database
4. **Display**: Users can browse, filter, and search through curated opportunities

## 📝 Usage

- Browse all available tech jobs
- Use filters to narrow down by job category
- Search for specific roles, technologies, or companies
- Sort by most recent postings
- Click on any job to view full details
